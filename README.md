## Deskripsi

Aplikasi berita sederhana untuk Android. Menampilkan berita terkini dari berbagai sumber dengan kategori yang dapat disesuaikan. Dibangun dengan Kotlin dan Retrofit untuk konsumsi REST API.

## Tech Stack

- **Kotlin** - Bahasa pemrograman utama
- **Android SDK** - Platform development
- **Retrofit** - HTTP client untuk REST API
- **MVVM Architecture** - Pola arsitektur
- **Glide** - Image loading

## Fitur Utama

- Tampilkan berita terkini dari berbagai sumber
- Filter berita berdasarkan kategori
- Detail berita dengan gambar dan konten lengkap
- Bookmark berita favorit
- Share berita ke media sosial

## Instalasi

1. Clone repository
```bash
git clone https://github.com/RanggaNf/news.git
```
2. Buka project di Android Studio
3. Sync Gradle dan jalankan di emulator/device
